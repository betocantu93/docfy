# @docfy/ember

Ember integration with Docfy.

## Compatibility

- Ember.js v3.16 or above
- Ember CLI v2.13 or above
- Node.js v10 or above

## Installation

```
ember install @docfy/ember
```

## Usage

Refer to the [documentation site](https://docfy.dev/docs/ember).
